FAQ

How many internal links should each page have?

Aim for 3–8 internal links per page (1 per 200–300 words of content). Pillar pages should receive links from all cluster/blog content related to their topic.

What is a pillar page?

A pillar page is a comprehensive, authoritative piece covering a broad topic. It links out to and receives links from more specific cluster content, forming a topic cluster structure.

What anchor text should I use for internal links?

Use descriptive, keyword-rich anchor text that tells both users and search engines what the linked page is about. Avoid generic "click here" or "read more" anchors.
