FAQ

What is landed cost?

Landed cost is the total cost of a product once it reaches your warehouse — product price plus freight, insurance, customs duty, taxes and clearing charges.

Is IGST part of landed cost?

IGST paid on imports is generally available as input tax credit, so it is often excluded from the costing base. This tool shows landed cost both with and without IGST.

How is customs duty calculated?

Basic customs duty is charged on the CIF value (cost + insurance + freight). Actual duty depends on the HS code of the product.
