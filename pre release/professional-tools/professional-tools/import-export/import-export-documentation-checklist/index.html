FAQ

What is an IEC code and who needs it?

Importer Exporter Code is a 10-digit number from DGFT, mandatory for any business importing or exporting goods from India. One-time registration.

What is a Certificate of Origin?

A COO certifies the country where goods were produced. Required for customs clearance, FTA preferential duty claims, and importing country compliance.

What happens if documents are missing at customs?

Missing documents lead to demurrage, customs hold and potential auction of goods. Having all documents ready prevents costly delays.
