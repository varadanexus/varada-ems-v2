FAQ

How is SIP maturity calculated?

FV = P × (((1+i)^n − 1) ÷ i) × (1+i), where P is the monthly amount, i the monthly return (annual ÷ 12 ÷ 100) and n the number of months.

Are SIP returns guaranteed?

No. SIPs invest in market-linked instruments, so returns fluctuate. The figure here uses a constant assumed return for illustration only.

Does a higher duration help?

Yes, longer durations benefit more from compounding, so gains grow faster in later years.
