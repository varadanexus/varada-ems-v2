FAQ

How many bricks per square foot?

About 5 bricks/sq ft for a 4.5-inch (half-brick) wall and about 10/sq ft for a 9-inch (full-brick) wall.

How much mortar is needed?

Mortar is roughly 20–25% of the wall volume; this tool estimates the cement and sand for it.

Should I add wastage?

Yes, add about 5% for breakage and cutting.
