FAQ

How is schedule variance measured?

Compare actual work done against the planned progress for the time elapsed; a negative gap means the project is behind.

What does projected completion mean?

Extrapolating the current pace: elapsed months ÷ fraction of work done estimates the total months needed.

Is linear planned progress realistic?

It is a simplification; most projects follow an S-curve, slower at start and finish.
