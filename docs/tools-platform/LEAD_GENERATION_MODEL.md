# Lead Generation Model

## Objective

Turn tool usage into qualified demand without reducing the tool’s standalone usefulness.

## Funnel Model

### Top of Funnel

- organic search traffic
- tool discovery pages
- sector landing pages

### Middle of Funnel

- tool usage
- contextual educational guidance
- result interpretation

### Bottom of Funnel

- audit request
- consultation request
- estimate request
- implementation inquiry

## CTA Types

- Book a consultation
- Request a custom estimate
- Get an audit
- Speak to an expert
- Ask for implementation support

## CTA Triggers

- high-complexity result
- compliance gap detected
- cost threshold exceeded
- project scale indicates consulting need
- uncertainty or risk detected in inputs

## Lead Data to Capture

- name
- company
- sector
- email / phone
- project scale
- timeline
- service need

## Guardrails

- Do not block results behind forms.
- Do not use generic CTAs disconnected from tool output.
- Do not ask for excessive information too early.
