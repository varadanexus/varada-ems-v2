# Tools Platform

This documentation tree defines the Varada Nexus Tools Platform as a separate initiative from EMS.

## Purpose

- Build non-AI tools based on formulas, rules, structured inputs, and checklists.
- Attract SEO traffic through useful, high-intent utility pages.
- Convert visitors into qualified leads for Varada Nexus services.

## Scope

- Public-facing tools
- Tool strategy and business model
- Content, SEO, QA, and specification standards
- Reusable templates and role-based documentation agents

## Guardrails

- Do not duplicate EMS documentation.
- Reference EMS documents by path only when needed.
- Treat EMS architecture and historical documentation as separate and protected.
