# Tool Categories

## Primary Categories

### 1. Calculators

Formula-driven tools that estimate cost, time, quantity, staffing, margin, or ROI.

### 2. Checklists

Step-based readiness or compliance tools that help users evaluate status and gaps.

### 3. Assessments

Rule-based scoring tools that classify maturity, fit, risk, or readiness.

### 4. Estimators

Structured-input tools that produce approximate business, project, or operational estimates.

### 5. Planning Tools

Tools that help users organize scope, resources, process steps, or documentation requirements.

### 6. Validators

Tools that check completeness, consistency, or rule compliance against known criteria.

## Category Fit Rules

- Use calculators for numeric outputs.
- Use checklists for readiness and compliance.
- Use assessments for scored decisions.
- Use estimators for planning with assumptions.
- Use validators for pass/fail or issue detection.

## Exclusions

- No AI chat tools
- No generative text tools as the core value
- No opaque scoring without documented logic
