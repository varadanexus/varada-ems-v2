# First 25 Tools Roadmap

This roadmap defines the first batch of non-AI, rule-based Tools Platform opportunities for Varada Nexus.

Scoring intent:

- Prioritize SEO traffic potential.
- Prioritize lead-generation relevance.
- Prefer deterministic tools with low to moderate implementation complexity.
- Use Indian business context where useful.
